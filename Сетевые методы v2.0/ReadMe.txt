Программа для автоматизации расчёта сетевых методов планирования табличным и матричным способами. Переписано на C#, скорость работы увеличена в несколько раз. Программа умеет проводить построение графика работ и его оптимизацию. Перебираются все возможные варианты, поэтому время оптимизации увеличивается в геометрической прогрессии с ростом некритических переходов между операциями. Если переходов много, то есть вероятность, что расчёт так и не завершится.

Рекомендуется для каждого нового расчёта перезапускать программу (есть баги, пока не отловил).

Для работы требуется .NET, желательно последний.
